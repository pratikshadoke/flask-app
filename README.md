# 🚀 InfraWatch - DevOps Dashboard

A modern DevOps monitoring dashboard built using Flask, Docker, Kubernetes, and GitHub Actions.

## 🔧 Tech Stack
- Flask
- Docker
- Kubernetes
- Nginx
- GitHub Actions

## 🚀 Run Locally
docker-compose up --build

## ☸️ Deploy to Kubernetes
kubectl apply -f k8s/

## 🌐 Features
- System Monitoring (CPU, Memory, Disk)
- Real-time Dashboard
- CI/CD Pipeline
- Containerized Deployment
